# Exercise 1 - Part 1 Boolean Logic

Write down what the following statements will return. Try to figure this out before putting the commands in the chrome console.

1. 2 == "2"
   * Answer: True.
2. 2 === 2
    * Answer: True.
3. 10 % 3
   * Answer: 1
4. 10 % 3 === 1
   * Answer: True
5. true && false
   * Answer: False
6. false || true
   * Answer: True
7. true || false
   * Answer: True
